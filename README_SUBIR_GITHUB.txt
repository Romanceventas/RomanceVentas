SUBIR A GITHUB PAGES

Archivos que debes subir al repositorio:
- index.html
- manifest.json
- sw.js
- .nojekyll
- carpeta icons/

Estructura:
/ index.html
/ manifest.json
/ sw.js
/ .nojekyll
/ icons/icon-192.png
/ icons/icon-512.png

Si editas el catálogo desde el panel admin y descargas un nuevo index.html, solo reemplaza ese archivo.
Los demás archivos se mantienen.

Luego haz commit y push.
GitHub Pages puede tardar unos minutos en reflejar cambios.
